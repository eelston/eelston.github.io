Pomodoro Timer To-do List, Take Two

Summer 2024 | Personal Project | Language: C

This project is the second attempt of a project I began for ECE 287 in Fall 2024. Originally, I was using a Raspbery Pi Pico W with a graphic LCD and a 4-digit 7-segment display, programming in CircuitPython before switching to MicroPython. I didn't complete the initial project within the semester, or the semester after, but I didn't want to completely abandon the idea. I returned to the project in August. I first tried to continue where I left off, but after some continued difficulties with the GLCD, I decided that I might do better with a different approach.

Components

  • Atmega328p on Arduino UNO
  • 128x64 OLED display
    • 12C interface
    • SSD1306 display driver by Preston Sundar
  • 3 pushbuttons
  • Breadboard
